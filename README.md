# aspnetcore-mock-file-upload-server

Mock file upload server as a companion to [NgRx File Upload](https://github.com/wesleygrimes/ngrx-file-upload)

## Prerequisites

- Install [.NET Core](https://dotnet.microsoft.com/download)

## Start

```bash
# Build
dotnet build

# Start Server
dotnet run
```
